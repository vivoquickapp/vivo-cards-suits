# 【组件】textarea 组件

# 描述

textarea 独占一块区域，此区域内的文本可以换行显示，超过显示区域后省略多余内容。

# 使用效果

# 使用方法

在.ux文件中引入组件

<import
  name="v-textarea"
  src="vivo-cards-suits/components/jovi/components/textarea"
></import>

<template>
  <div class="card">
    <v-textarea lines="7" text="{{text}}"></v-textarea>
  </div>
</template>

        愉悦体验的智能产品，并将敢于追求极致、持续创造惊喜作为vivo的坚定追求`
      };
